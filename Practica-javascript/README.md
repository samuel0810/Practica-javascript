# Practica-javascript - Practica usando javascript

Crear una página web interactiva cuya finalidad sea preguntar al usuario su nombre y año de nacimiento, calcular su edad y mostrar la información en el navegador. Al abrir la página, debería aparecer un cuadro de diálogo solicitando estos datos. El nombre y la edad calculada deben presentarse de forma clara y agradable en la pantalla.

Por ejemplo, si el nombre del usuario es Anna y nació en 1990, la página debe mostrar:
"Hola, Ana. Tienes 34 años."  (asumiendo que el año actual es 2025).

Utilice JavaScript para manejar la interacción y realizar los cálculos necesarios. Diseña la página con un encabezado que explique brevemente su función, como "Calculadora de edad", y si quieres puedes usar estilos CSS para hacerla más atractiva visualmente. Puede utilizar colores de fondo suaves, fuentes legibles y centrado en el contenido para que la presentación sea más agradable.
Objetivos de aprendizaje

Al completar esta tarea, usted podrá:

    Cree una página web interactiva con JavaScript para manejar la entrada del usuario y realizar cálculos
    Diseña una página web con un encabezado y utiliza estilos CSS para mejorar su atractivo visual
